Primeiros passos

Fácil de usar

A utilização do pacote é extremamente fácil, já que está disponível no gerenciador de pacote pip. Assim, sua instalação e execução são simplificadas, tornando o processo muito acessível para qualquer usuário.

Desenvolvido em python

O AISP é desenvolvido em Python, uma linguagem amplamente utilizada na comunidade de aprendizado de máquina. Logo, é possível a utilização com diversos pacotes que facilitam o trabalho com os dados.
