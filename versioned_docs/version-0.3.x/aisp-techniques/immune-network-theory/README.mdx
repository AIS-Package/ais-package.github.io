# Immune Network Theory

This technique was introduced by **Niels Jerne (1974)** and models the immune system as a dynamic network,
in which cells and molecules are capable of recognizing each other. [1](#1)

---

The Artificial Immune Network can be applied in different contexts, such as:
- **Clustering**
- **Optimization**
- **Classification**
